## Usage

### GUI

```bash
python gui.py
```

* Scans **do not run automatically**
* Click **Run Scan** to execute checks
* Enable online checks explicitly if desired

### CLI

```bash
python vault_check.py
python vault_check.py --online
python vault_check.py --json
```

---

## USAGE NOTES

* Online intelligence checks are **disabled by default**
* HIBP password checks never transmit full passwords or hashes
* Advanced scanning options are placeholders for future releases

---

## SECURITY & PRIVACY

* No system data is uploaded automatically
* No credentials are stored
* No background processes
* No persistent agents

You are always in control of what runs and when.
